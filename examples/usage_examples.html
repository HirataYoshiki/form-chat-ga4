Contact Form Widget Examples

This page demonstrates how to embed the contact form widget on your website. The widget loader script (form_widget_loader.js) will read the data-* attributes from the script tag and create a placeholder element where the actual form will be rendered (or indicate what will be rendered).

To use the widget, include the form_widget_loader.js script in your HTML and provide the necessary data-* attributes.

Minimal Configuration

This example uses only the required data-form-id attribute. The loader script will use default values for other settings like title, theme, and API endpoint.

Code:

<script src="form_widget_loader.js" data-form-id="default-contact-form"></script>

Customized Configuration

This example demonstrates using various data-* attributes to customize the widget's appearance and behavior.

Code:

<script src="form_widget_loader.js" 
        data-form-id="product-inquiry-001"
        data-api-endpoint="https://your-custom-api.com/submit-inquiry"
        data-theme="dark"
        data-position="bottom-left"
        data-title="Product Inquiry"
        data-button-text="Send Inquiry"></script>

Important: For these examples to function correctly, the form_widget_loader.ts file (located in the same examples directory) must be compiled into JavaScript as form_widget_loader.js and be accessible at the same path relative to this HTML file. The current loader script only creates a placeholder div and logs attributes to the console; it does not yet render the actual React form widget.
